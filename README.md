- [Axios](https://github.com/axios/axios)

## :rocket: Como Usar:

```bash
# Clone este repositório e acesse sua pasta
$ git clone https://github.com/evertoont/desafio-reactjs && cd desafio-reactjs

# Instale as dependências
$ yarn install

# Iniciie a aplicação
$ yarn start
```
